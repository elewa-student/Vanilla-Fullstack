To use my RESTful Notes API inspect this page and use the "api_connection" object from the console.

DOCS:
: Welcome page -> .get([""])
: Reset DB -> .post(["reset"])
: View all notes -> .get(["all"])
: Create note -> .get(["add"])
: Create note -> .post(["add"], "note")
: Read note -> .get([":id"])
: Update note -> .get([":id", "update"])
: Update note -> .post([":id", "update"], "note")
: Delete note -> .get([":id", "delete"])
: Delete note -> .post([":id", "delete"])
